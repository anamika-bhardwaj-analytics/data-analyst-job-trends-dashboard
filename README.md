# Data Analyst Job Market Dashboard (Power BI)

## 📊 Overview
This project presents an interactive Power BI dashboard analyzing data analyst job postings across India to uncover trends in hiring demand, required skills, salary distribution, and work modes.

## 🎯 Objective
To analyze the data analyst job market and identify key trends in skills demand, hiring locations, experience levels, and compensation to support data-driven career insights.

## 🚀 Key Insights
- Total Job Listings: 500  
- Average Salary: ₹1.11M  
- Remote Job Listings: 803  
- Top Hiring Locations: Pune (291), Noida (288), Delhi NCR (278), Chennai (274), Kolkata (273)  
- Most In-Demand Skills: SQL, Tableau, Python, Power BI  
- Highest Demand by Experience Level: Entry Level (1094 roles), followed by Mid Level (847)  
- Top Hiring Companies: Microsoft (177), Wipro (173), Reliance (164), PwC (163), TCS (146)  
- Work Mode Distribution: Remote (35.96%), Hybrid (34.98%), Onsite (29.06%)  
- Peak Job Posting Trend observed towards the end of the month (~110 listings)

## 📈 Features
- KPI cards highlighting total jobs, salary trends, and remote opportunities  
- Location-wise job distribution analysis  
- Time-based job posting trends  
- Skills demand visualization  
- Experience-level demand breakdown  
- Company-wise hiring analysis  
- Work mode comparison (Remote vs Hybrid vs Onsite)  
- Interactive filters for dynamic exploration  

## 🛠 Tools & Skills Used
